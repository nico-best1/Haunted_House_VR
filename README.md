# Haunted House VR

**Haunted House VR** is an immersive virtual reality horror experience developed with **Unity**.  
Explore a mysterious abandoned mansion full of secrets, scares, and an eerie atmosphere, created to immerse the player in a short but intense story.

---

## Main Features

- Fully immersive **virtual reality** experience  
- Detailed environments with dynamic lighting  
- 3D ambient sound  
- Interaction with physical objects through VR controllers  
- Visual and audio effects designed to create psychological tension  

---

## Technologies and Tools Used

- **Engine:** Unity, version 6000.2.9f1  
- **Programming Language:** C#  
- **VR SDK:** OpenXR  
- **Audio:** FMOD  

---

## Controls and Interaction

- **Movement:** Continuous using joystick  
- **Interaction:** Press trigger or grip button
